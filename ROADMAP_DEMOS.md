# Roadmap de demos

## Base actual

- `index.html`: landing comercial principal de LocalConnect.
- `abogados/`: demo sobria orientada a confianza y captacion.
- `restaurante/`: demo emocional orientada a atmosfera y reservaciones.
- `shared/demo.js`: helpers compartidos de interaccion para demos.

## Regla para nuevas plantillas

Cada nueva demo debe demostrar dos cosas:

1. Un giro especifico.
2. Un nivel de producto claro: `Profesional` o `Premium`.

## Diferencia entre niveles

### Web Profesional

- 4 a 6 secciones.
- Enfoque en claridad comercial.
- CTA directo.
- Pruebas de confianza basicas.
- Formularios o WhatsApp sin demasiada complejidad visual.

### Web Premium

- 6 a 8 secciones.
- Direccion de arte mas fuerte.
- Storytelling y ritmo visual mas trabajado.
- Mayor profundidad comercial.
- Mas piezas de confianza, composicion y detalles de experiencia.

## Siguiente orden recomendado

1. `consultorio-profesional`
2. `inmobiliaria-premium`
3. `constructora-profesional`
4. `hotel-boutique-premium`
